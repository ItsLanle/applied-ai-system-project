# Assets Folder

This folder contains visual assets for the BugHound project.

## Architecture Diagram

- `bughound_architecture.mmd`: Mermaid markup for the BugHound agent architecture flowchart.

To create a PNG image:
1. Copy the content of `bughound_architecture.mmd`
2. Paste it into the [Mermaid Live Editor](https://mermaid.live)
3. Export as PNG and save as `bughound_architecture.png` in this folder

## Screenshots

Add screenshots of the Streamlit app here, such as:
- `app_main_screen.png`: Main interface showing code input
- `app_analysis_results.png`: Analysis results with issues detected
- `app_evaluation_report.png`: Evaluation harness output

To capture screenshots:
1. Run the Streamlit app: `streamlit run bughound_app.py`
2. Navigate to different sections
3. Use your system's screenshot tool (Cmd+Shift+4 on Mac, Snip & Sketch on Windows, etc.)
4. Save images here with descriptive name


## Usage in Documentation

Reference images in your README.md like this:
```markdown
![BugHound Architecture](assets/bughound_architecture.png)
![Main Screen](assets/app_main_screen.png)
```
